Workout tracker application:
1- Create a google sheet. and sheety project 
2- Create an account on nutritionix api and get the api key(Estimate calories burned for various exercises using natural language.)
3- this application gets input from user and post a request to nutritionx api to get a json as the response and then update the google sheets with that information.
Tech used in this projects:
python, pycharm, nutitionix api, google sheet ans sheety which connects google sheet to our application.
